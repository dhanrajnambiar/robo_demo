Robo Operating Manual

This is to simulate AJAX POST requests to Backend API's for creation of ROBOT, and navigate it.

Instruction to create a ROBO

Kindly open web console or use POSTMAN. Make an ajax call like below or POST request to API using POSTMAN.

$.ajax({ url:"/robo/create_robo", type:'POST', data:{ "name":"Robo_1" }, success: function(data){ console.log(data); } });

Note : The ROBO Name you pass to API must be unique and not reused.

Commands to ROBO

Make an ajax call like belowor POST request to API using POSTMAN.

$.ajax({ url:"/robo/move", type:'POST', data:{ "name":"Robo_1", "command":"PLACE 2,2,EAST MOVE MOVE REPORT" }, success: function(data){ console.log(data); } });

Note : The ROBO Name you pass to API must be unique and not reused.
